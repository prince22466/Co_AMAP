# Co_AMAP

Co_AMAP is a workspace for collaborating with AI (primarily Codex) across many types of technical tasks.

## Current areas
- `Co_ML` — machine learning experiments.
- `Co_Kaggle` — Kaggle game/project work.
- `Co_Quantum` — quantum-related experiments.
- `Co_PDE` — partial differential equation work.

## Goal
Build and iterate on as many practical, AI-assisted projects as possible in a single repository.
